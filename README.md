node.js
====================

Info
--
Node.js server to get distance using Google Map, and query sites from the elasticSearch


Install
--
Install Node.js

Install extensions using npm on the Node.js command line

* npm install fs
* npm install http
* npm install googlemaps
* npm install path
* npm install request
* npm install url
